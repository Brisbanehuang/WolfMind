# 狼人杀游戏 - 前端

基于 React + TypeScript 的狼人杀游戏前端界面。

```
frontend/

│   ├── index.html
│   └── favicon.ico
├── src/

│   ├── pages/               # 页面
```

## 快速开始（待实现）

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 构建生产版本

```bash
npm run build
```

## 技术栈（计划）

- **框架**: React 18
- **语言**: TypeScript
- **构建工具**: Vite
- **状态管理**: Redux Toolkit
- **UI 组件**: Ant Design / Material-UI
- **实时通信**: WebSocket
- **HTTP 客户端**: Axios
- **路由**: React Router
